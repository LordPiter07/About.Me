Acerca de este Sitio Web

Este es un portafolio muy sencillo compuesto de 4 páginas, maquetado con HTML y estilizado con CSS, con el objetivo de servir de presentacion para el curso FullStack Java del programa Codo a Codo 4.0. Este sitio es full responsive, adaptable a cualquier pantalla.
Para diseñar este proyecto me basé en el portafolio de Adam Matt, me gusto el estilo minimalista y simple y basado en eso hice mi própio bosquejo.
Te recuerdo que podes consultar todo el codigo del sitio en mi pagina de Github.
